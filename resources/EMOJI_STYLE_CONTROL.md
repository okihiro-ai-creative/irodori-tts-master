# EMOJI_STYLE_CONTROL

Irodori-TTSは入力テキスト内の絵文字で、話し方、感情、非言語音、環境感を制御できる。
ただし効果は文脈依存であり、完全な制御ではない。

## 使い方

- 本文の該当箇所に絵文字を挿入する
- 同じ絵文字を複数回使うと効果が強まる場合がある
- 最終出力用の本文には、効果が破綻しない範囲で少量だけ使う

## 代表カテゴリ

| 用途 | 指示内容 |
|---|---|
| 囁き・近距離 | 囁き、耳元、近い音 |
| 呼吸 | 吐息、溜息、寝息、息切れ |
| 間 | 間、沈黙、ゆっくり |
| 笑い | くすくす笑い、含み笑い |
| 泣き | 嗚咽、泣き声、悲しみ |
| 強い感情 | 叫び、怒り、驚き、苦しさ |
| 体調音 | 咳、鼻すすり、くしゃみ、咳払い |
| 口元音 | リップノイズ、咀嚼音、水音 |
| 環境 | 電話越し、スピーカー越し、エコー |
| 進行 | 早口、ゆっくり、ナレーション、独白 |

## 正例

```text
あはは、それは少し言いすぎですよ。まあ、気持ちはわかります。
```

```text
ごめん、今日は少し声が出にくくて。少しだけ休ませてください。
```

## 反例

```text
こんにちは！！！！！！！！！！！！！！！！！！！！！！！！！
```

絵文字や記号を過剰に詰めると、音声が不安定になる。

## スキル内での扱い

- 絵文字は `outputs/normalized_text.md` に明示する
- どの効果を狙ったかを `outputs/inference_plan.md` に記録する
- 失敗時は絵文字を減らす、caption側に移す、seed候補を変える、の順で調整する
