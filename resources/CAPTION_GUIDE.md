# CAPTION_GUIDE

VoiceDesign caption は、声の人格・演技・距離感・環境を短く具体化する。
captionは長く複雑にしすぎると追従が落ちるため、原則2〜4文に収める。

## 推奨構造

```text
{年齢・性別・声の高さ}。{感情・距離感・話速・口調}。{用途や場面に合う演技指示}。
```

## 指定できる要素

| 要素 | 例 |
|---|---|
| 年齢・性別 | 若い女性 / 落ち着いた大人の男性 |
| 声の高さ | 少し高め / 中低音 / 低く深い |
| 距離感 | 近い距離感 / 標準的な距離 / 電話越し |
| 感情 | 穏やか / 喜び / 不安 / 怒り / 悲しみ |
| 話速 | ゆっくり / 自然 / 早口 |
| 口調 | 丁寧 / 親しい / 講師風 / ナレーション |
| 環境 | 静かな室内 / スタジオ収録 / 反響のある空間 |

## 正例

```text
落ち着いた大人の男性。低めで深く響く声で、聞き手に安心感を与えるように丁寧に話す。ビジネス解説のナレーションとして自然な距離感。
```

```text
若い女性の声。少し高めで明るく、近い距離感で親しみやすく話す。テンポは自然で、聞き手に寄り添う案内役の口調。
```

## 反例

```text
高い声で低い声で、子どもっぽく大人っぽく、怒りながら穏やかに、遠くから耳元で話す。
```

これは矛盾が多く、VoiceDesignの条件衝突を起こす。

## 参照音声ありの場合

参照音声ありでは、captionは声質を上書きするためではなく、感情・話し方・環境を補助する。

正例:
```text
参照音声の自然な声質を保ったまま、穏やかで丁寧に話す。少し近い距離感で、聞き手に安心感を与える。
```

反例:
```text
低い男性の参照音声を、高い少女声に変える。
```

これは参照音声とcaptionが矛盾するため禁止。
